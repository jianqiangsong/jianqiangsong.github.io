git教程笔记

撰写于 2018-08-21 修改于 2018-08-23 标签 git

查看当前分支下的修改文件状态

git status

修改文件后查看修改地方

git diff  1.txt

查看最近提交的记录

git log     git log –pretty=oneline  (美化显示)   git reflog  (显示修改的版本号)

回退到上一个版本

git reset  –hard HEAD^  |  git reset  –hard 版本号    git reset  –hard 6fcfc89

撤销修改 git checkout — file 可以丢弃工作区的修改，未添加到暂存区的 没有add会撤销

git checkout –readme.txt

删除后没有提交commit 恢复文件

git checkout  --b.txt

添加远程仓库地址

git remote add origin https://github.com/testgit.git

推送到仓库主分支

git push origin master

创建dev分支，然后切换到dev分支上

git checkout -b dev      命令加上 –b参数表示创建并切换

查看分支

git branch查看分支，会列出所有的分支，当前分支前面会添加一个星号

主分支上 git merge命令用于合并指定分支到当前分支上

git checkout master    git merge dev

移除分支：git branch –d name

git branch –d name

git修改远程仓库地址 

git remote origin set-url [url]

two

git remote rm origin   
git remote add origin [url]

查看远程仓库信息 

git remote -v

命令 git remote show [remote-name] 查看某个远程仓库的详细信息,比如要看所克隆的origin 仓库

git remote show origin
  1. git init
  2. git add README.md
  3. git add .
  4. git commit -m “first commit”
  5. git remote add origin https://github.com/userName/repoName.git
  6. git push –force origin master
  7. git push origin master –force 强制推送数据到服务器

获取主分支

  1. git fetch origin master
  2. git rebase tmp // 修复回退之前版本 git rebase –abort
  3. git push origin HEAD:master
  4. git branch -D tmp
  5. git commit -a -m “str” git commit -am “str” //添加并提交
