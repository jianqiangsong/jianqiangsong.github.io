redis-cluster 集群模式报错 知识点汇总 ：

在redis3.0之前 redis服务不支持集群模式,只能通过客户端分片
redis事物支持不完整 不支持回滚
redis3.0 可以集群 特点一致性hash crc32算法

redis-cluster
-

  1. redis官网推出，可线性扩展到1000个节点
  2. 无中心架构
  3. 一致性哈希思想
  4. 客户端直连redis服务，免去了proxy代理的损耗

Redis适用场景
-

  • KV存储
  • 缓存（ttl LRU…）
  • Lru是redis达到maxmemory之后，可选的key删除策略。
  • 消息中间件
  • 分布式锁
  • ……

#ubnatu 16.04 安装redis-server服务端 

sudo apt-get install  redis-server

安装完成会自动启动服务

ps -aux|grep redis |或者  netstat -nlt|grep 6379

通过redis-cli 进入客户端

redis的配置文件 

sudo vi /etc/redis/redis.conf

让Redis服务器被远程访问
默认情况下，Redis服务器不允许远程访问，只允许本机访问，所以我们需要设置打开远程访问的功能。
用vi打开Redis服务器的配置文件redis.conf

sudo vi /etc/redis/redis.conf
注释bind
#bind 127.0.0.1

重启redis 服务

sudo /etc/init.d/redis-server restart

#ubnatu 16.04 安装php-redis 操作客户端扩展

下载phpredis扩展文件，如果服务器没有安装git服务，下载之前，首先安装一下git服务，这里不做描述，自行百度

  1. git clone https://github.com/phpredis/phpredis.git
  2. 移动文件
    mv phpredis(此处是你clone下的文件) /etc/phpredis

  3. 安装
    cd /etc/phpredis

  4. 如果phpize命令没有响应，可能是没有安装php-dev。我目前安装的是php7.0，键入命令
    sudo apt-get install php7.0-dev

  5. 然后再phpize编译依次键入命令
    phpize
  6. ./configure
  7. make
  8. make install
  9. 修改配置文件
  10. vim /etc/php/7.0/apache2/php.ini
  11. /php_shmop（查找字符串）
  12. extension=/etc/phpredis/modules/redis.so

#ubnatu 16.04 安装memcached服务

全局安装服务 

sudo apt-get install memcached

启动服务：

memcached -d -m 128 -p 11111 -u root
  • -p 监听的端口
  • -l 连接的IP地址, 默认是本机
  • -d start 启动memcached服务
  • -d restart 重起memcached服务
  • -d stop|shutdown 关闭正在运行的memcached服务
  • -d install 安装memcached服务
  • -d uninstall 卸载memcached服务
  • -u 以的身份运行 (仅在以root运行的时候有效)
  • -m 最大内存使用，单位MB。默认64MB
  • -M 内存耗尽时返回错误，而不是删除项
  • -c 最大同时连接数，默认是1024
  • -f 块大小增长因子，默认是1.25-n 最小分配空间，key+value+flags默认是48
  • -h 显示帮助

#Ubuntu 下安装 PHP7 扩展 Memcached

安装依赖

sudo apt-get install memcached

sudo apt-get install libmemcached11 libmemcached-dev libmemcachedutil2

#安装 memcached 扩展

查看 php7.-0 的软件包

  1. apt-cache search -n php7.0
  2. 下载 memcached 到本地仓库，并切换到 php7 版本
  3. git clone https://github.com/php-memcached-dev/php-memcached.git
  4. cd php-memcached/
  5. git checkout php7
  6. 执行配置文件，注意：这一步可能会出错，原因可能是缺少 memcached 依赖，　安装上面安装便可
  7. ./configure –disable-memcached-sasl
  8. 编译安装
  9. make && sudo make install
  10. 查看 memcached 插件是否被添加
  11. /etc/php/7.0/mods-available/memcached.ini
  12. 重启 php

  13. sudo systemctl restart php7.0-fpm

    执行 phpinfo.php 查看 memcached 是否被添加
