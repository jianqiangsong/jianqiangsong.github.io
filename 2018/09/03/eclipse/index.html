eclipse for php IDE使用工具快捷键汇总

注意 当存在桌面第三方应用或者是某个单个项目配置 可能导致编辑全局失效

  1. 定位到某一行 CTRL + L
  2. 复制当前行 CTRL +ALT + DOWN 往下
  3. 复制当前行 CTRL +ALT + UP 往上
  4. 全局查找 CTRL + H
  5. 打开项目属性 ALT + ENTER
  6. 快速显示当前类的继承结构 Ctrl+T
  7. 最大化当前的Edit或 View (再按则反之) Ctrl+M
  8. 快速显示 OutLine Ctrl+O 编辑器部分显示详细列表
​Ctrl+K 参照选中的Word快速定位到下一个

正向增量查找(按下Ctrl+J后,你所输入的每个字母编辑器都提供快速匹配定位到某个单词,如果没有,则在stutes line中显示没有找到了,查一个单词时,特别实用,这个功能Idea两年前就有了
Ctrl+J

附上 redis 服务启动后台守护进程的

  1. 进入 DOS窗口
  2. 在进入redis的安装目录
  3. 输入：redis-server –service-install redis.windows.conf –loglevel verbose ( 安装redis服务 )
  4. 输入：redis-server –service-start ( 启动服务 )
  5. 输入：redis-server –service-stop （停止服务）
